# 🚀 AI Resume Builder

An AI-powered Resume Builder web application that helps users create professional, ATS-friendly resumes in minutes.

---

## 🌐 Live Demo

👉 https://ai-resume-bolt.netlify.app

---


## ✨ Features

- 🔐 JWT-based Authentication (Signup/Login)
- 📝 One-input Resume Creation (single paragraph input)
- 🤖 AI-powered Resume Generation using Google Gemini API
- ✏️ Editable Resume Sections (Experience, Education, Skills, Summary)
- 👀 Live Resume Preview
- 📄 Download Resume as PDF
- 📱 Fully Responsive UI

---

## 🧠 How It Works

1. User signs up / logs in
2. Creates a new resume
3. Enters all details in a single paragraph
4. Clicks **"Generate with AI"**
5. AI converts input into structured resume
6. User edits and customizes
7. Downloads final resume as PDF
